# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.1.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.1.0] - 2025-01-11

### Added
- Explicit trigger patterns section (IDEA:, "save this idea", etc.)
- Clear session start conditions (when to read SESSION.md)
- Good vs Bad usage examples with anti-patterns
- Integration guidance with TodoWrite, brainstorming, writing-plans skills
- Implicit trigger documentation

### Changed
- Section names from Italian to English (Oggi→Today, Prossimo→Next, Storico→History)
- Workflow diagram now includes SESSION.md existence check
- Improved description in YAML frontmatter

## [1.0.0] - 2025-01-11

### Added
- Initial public release
- SKILL.md with complete workflow documentation
- IDEAS-example.md template with category tables
- SESSION-example.md template with handoff structure
- Parking Lot section for off-topic ideas
- Idea lifecycle tracking (Completed/Superseded/Rejected)
- Source tracking (User/Discussion/Claude)
- Effort/Value matrix (S/M/L)

### Changed
- Clarified that this is for project-related ideas, not generic notes
- Templates renamed to examples (not rigid structures)
- Fixed character encoding issues (arrows, special chars)
- Replaced dot diagram with ASCII art for GitHub compatibility
